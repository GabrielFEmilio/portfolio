    Transparência e privacidade

    Políticas do site

    Última atualização: 17 de junho de 2026.

    Responsável e contato

    Este site-portfólio é mantido por Gabriel Feltrin Emilio. Para solicitações sobre privacidade, dados pessoais, correções ou remoções, use o e-mail: gabriel.emilio.dev@gmail.com.

    Dados e finalidades

    • Preferência de idioma: quando você troca o idioma, o site salva a escolha no navegador usando localStorage, na chave site_lang, para lembrar a preferência nos próximos acessos.
    • Formulário de contato: se você enviar uma mensagem, nome, e-mail e conteúdo da mensagem são enviados ao serviço Formspree e depois encaminhados ao e-mail de contato, apenas para responder sua solicitação.
    • GitHub API: o site consulta dados públicos de repositórios e perfil do GitHub para exibir projetos e linguagens. Essa requisição pode transmitir dados técnicos comuns, como IP, navegador e horário da solicitação, ao GitHub.
    • Fontes e scripts externos: o site carrega Google Fonts e bibliotecas hospedadas em CDNJS/Cloudflare. Esses provedores podem receber dados técnicos da conexão necessários para entregar os arquivos.
    • Links externos: links para GitHub, LinkedIn, Gmail, Formspree ou sites publicados levam você para serviços de terceiros, que possuem políticas próprias.

    Cookies e tecnologias semelhantes

    O site não cria cookies próprios de publicidade ou analytics. A preferência de idioma é armazenamento funcional local. Serviços externos carregados pelo site ou acessados por links podem usar suas próprias tecnologias conforme as políticas deles.

    Direitos do usuário

    Você pode solicitar acesso, correção, exclusão, limitação, oposição ao tratamento ou informações sobre dados pessoais eventualmente enviados pelo contato. As solicitações devem ser feitas pelo e-mail informado acima.

    Retenção, segurança e transferências

    Mensagens de contato são mantidas apenas pelo tempo necessário para comunicação profissional e eventual proteção de direitos. Provedores como Formspree, Google, GitHub, LinkedIn e Cloudflare podem processar dados em outros países. O site busca limitar a coleta ao necessário, mas depende das práticas desses provedores.

    Menores de idade

    Este site não é direcionado a crianças e não solicita conscientemente dados de menores. Se um responsável identificar envio indevido de dados, pode pedir remoção pelo e-mail de contato.

    Limites deste aviso

    Este aviso aumenta a transparência, mas não substitui auditoria jurídica, contratos com operadores, revisão das configurações dos provedores, controles de retenção, nem eventual banner de consentimento se no futuro forem adicionados analytics, publicidade, pixels, rastreamento ou cookies não essenciais.
